# Cybersecurity Home Lab

## Overview
This project documents my personal cybersecurity home lab used to simulate attacks and analyze network traffic in a controlled environment.

## Objectives
- Practice cybersecurity and network defense
- Simulate attacks using Kali Linux
- Analyze network traffic with Wireshark
- Perform vulnerability scanning with Nmap

## Lab Architecture
The lab consists of multiple virtual machines connected in an isolated internal network.

Machines used:
- Kali Linux (Attacker)
- Windows (Victim)
- Ubuntu Server (Server)
- Monitoring Machine

## Tools Used
- Kali Linux
- Wireshark
- Nmap
- VirtualBox
- Linux

## Skills Demonstrated
- Network traffic analysis
- Vulnerability scanning
- Attack simulation
- Cybersecurity lab setup
